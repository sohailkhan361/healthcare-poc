
# Healthcare POC – Backend using ReactJs, TailwindCSS
This is the frontend for the Healthcare Plus Proof of Concept (POC) project.

# Installation
Install NodeJS in the system

# Go to the project directory
> cd frontend

# Install dependencies
> npm install
> npm i

# Set the .env file with the following
```
VITE_REACT_APP_API_BASE_URL="http://localhost:8000/api"
```
"VITE_" prefix is important here as VITE application recognizes env variables starting with "VITE_"

# Start the server
> npm run dev

# Tech Stack
ReactJs, ViteJs, TailwindCSS
